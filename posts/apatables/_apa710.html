In this 24-part series, each of the tables in Chapter 7 of the Publication Manual of the American Psychological Association (7th Edition) is recreated with apa7, flextable, easystats, and tidyverse functions.

The apa_cor function does a lot under the hood. By default, it gives the mean and standard deviation of each variable. By supplying a list of summary functions summary_functions = c("n", "M", "SD"), we can get the n for each variable, too.

# Make R symmetric and fill diagonal with 1s
R <- (R + t(R)) |> `diag<-`(1)

# Make data
d_simulated <- mvtnorm::rmvnorm(
  n = max(d$n), 
  mean = deframe(d |> select(Variable, M)),
  sigma = diag(d$SD) %*% R %*% diag(d$SD)) |> 
  as_tibble() |> 
  map2_df(d$n, \(x, n) {
    k <- length(x)
    x[sample(1:k, size = k - n)] <- NA
    x
  }) |> 
  rename_with(hanging_indent, width = 15, indent = 0)
